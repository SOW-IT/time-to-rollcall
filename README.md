https://sowwebsite.atlassian.net/wiki/spaces/dataandit/pages/211976194/Setting+up+TimeToRollCall+repository

## Cloud Functions Setup

1. npm install -g firebase-tools
2. firebase login
3. cd functions

### Deploy Cloud Functions

`npm run serve`

## Local Setup

1. cd frontend
2. npm install

### Run locally

(After cd frontend)
`npm run dev`
